# Next.js + TypeScript + Echo + Go のブログWebアプリケーション(フロントエンド側)

## Summary

Zennの記事やQiitaの記事も溜まっており、GitHubに管理しているMarkdownファイルを<br>
読み込んで表示するブログWebアプリケーション。<br>

まずは簡単なところから行う。<br>
